Style Guide

visual elements

Colours :

The colours for this portfolio are grounded in a predominantly black and white background. They are meant to be a source of focus and should thus be used sparingly. The colours should be derived from the content which they support to ensure cohesion between elements.

Only the main highlight colour should be used without concern for content related colours. The main highlight can be used for any highlight or interaction, however if there is a more suitable colour that has been derived from content being used then this colour should be used instead. The second and third highlight can be subject to change as the content they support changes.
